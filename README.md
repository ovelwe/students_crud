# Students CRUD

## Что сделано?

1. Разработана бэкенд часть для сайта-хранилища студентов и информации о них
2. Разработана простая верстка сайта
3. Добавлена возможность взаимодействия фронтенда и бэкенда

## Информация о проекте

**Backend**:

* Java
* Spring Boot
* lombok
* Hibernate
* PostgreSQL
* Maven
* Data JPA

**Frontend**:

* HTML
* CSS
* JavaScript
